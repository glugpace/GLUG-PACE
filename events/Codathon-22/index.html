CODATHON

GLUG PACE Codathon 2.0 is a continuity of an event that was conducted last year but with a fresh coders from pace. It will be a year long events that is conducted by glug pace 2021-22. This event is not restricted to any branch. This event is for anyone who wants to develop skill in programming, or anyone who wants to improve their present skill.

5 basic programs will be given you can seamlessly translate your concept into an executable solution using any programing language. It contains 3 rounds unraveled for this semester. Open for all branches who have a knack for programming.

Grading System

Grading will be done based on the number of test cases passed, and the efficiency of the algorithm. Hackerrank grading system will be implemented. On the event of tie, code readability will be checked by the committee.

The decision of the organizers in declaring the results will be final. The code should be an individual task, any type of malpractice will be immediately disqualified. Writing comments on your code will improve the chance of your winning.
