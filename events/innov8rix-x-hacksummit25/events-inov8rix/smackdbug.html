Smack-D-Bug

Fix it, beat it, and win it

Category:
CodeConquer

A two-phase debugging competition – spot errors in C code and fix frontend issues on a website.

Prerequisites:

  • C programming, HTML/CSS basics

Rules:

  • Mobile Phones and other electronic devices must be handed over to event coordinators to ensure fairness.
  • Use of internet is not allowed.
  • Any misuse of systems will lead to disqualification.
  • The event consists of 2 rounds.
  • Basic C programming and HTML/CSS buggy code will be provided.
  • In case of dispute, the ruling of the lead coordinator is final.

Judging Criteria:

  • Number of bugs fixed
  • Accuracy

Event Type: Team (2)

Date: 12th May

Time: 01:30 PM – 02:30 PM

Venue: Unix Lab
